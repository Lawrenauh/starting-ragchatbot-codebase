System Flow Explanation

  • User interacts with the Frontend Interface by entering a query
  • The query is sent via HTTP to the FastAPI Backend
  • The backend routes the query to the RAG System for processing
  • The RAG system uses Search Tools to retrieve relevant content from ChromaDB
  • Retrieved content is sent to Google Gemini for response generation
  • The AI-generated response is formatted and sent back to the Frontend
  • Session Manager maintains conversation context across interactions
